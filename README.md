# MOM Learn

A static educational website for learning *Plasmodium falciparum* blood-stage morphology and drug-exposure patterns.

## What is included
- Home page
- Morphology atlas
- Quiz mode
- Local progress tracking
- Mobile-friendly responsive layout

## How to run locally
Just open `index.html` in a browser, or serve the folder with any static web server.

Example:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## GitHub Pages
Upload the files in this folder to a GitHub repository and enable GitHub Pages from the repository settings.

## Replace the sample content
The current version uses simple placeholder visuals so the site can work immediately on GitHub Pages.
Replace the sample atlas cards and SVG quiz images in `app.js` with your real MOM image paths when ready.
